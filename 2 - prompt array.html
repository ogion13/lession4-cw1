Ввод массива из пользовательских данных и его суммирование

Напишите код который:

  • Запрашивает по очереди значения при помощи функции prompt и сохраняет их в массиве.
  • Заканчивает ввод, как только посетитель введёт пустую строку или не число или нажмёт «Отмена».
  • При этом ноль 0 не должен заканчивать ввод, это разрешённое число
  • Выводит сумму всех значений массива с помощью функции alert

Место для решения - инлайновый тег <script> на этой странице
